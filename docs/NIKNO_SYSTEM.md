# NIKNO — Creator Intelligence System

## Overview
NIKNO is designed as a Creator Operating System combining:
- AI intelligence layer
- creator dashboards
- funding workflows
- distribution + messaging

## Key Layers

### 1. Interface Layer
- Chat UI (SvelteKit)
- Conversational intelligence

### 2. Routing Intelligence
- Uses LLM Router
- Routes requests to specialized models

### 3. Data Layer
- MongoDB
- Stores users, content, messages, analytics

### 4. Expansion Layer
- Connect to Wix frontend
- Connect to Stripe / payments
- Connect to Supabase (optional scaling)

## Goal
Transform Chat UI into a full creator platform, not just chat.
